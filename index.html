Step 2

Copy the repository url from the forked repo

Step 3

Step 4

Step 5

Write git checkout -b `any desired word/words

Use your creativity here.
Step 6

Contribute to the code of this webpage

Step 7

In the terminal check git status then add the files you changed using git add `modified\add file name

Step 8

Write in the terminal, git commit -m `Your Message

Step 9

Write in the terminal, git push origin `Name you used in Step-4
