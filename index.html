Supreme-Lamp

-Exposure to Open Source

Supreme Lamp is a testing website. We believe everyone should gain some experience in contributing to websites. Feel free to contribute to this page. If you've written good code, then we will merge your code into the repository.

Follow the steps below in order to contribute

Step-1: Fork this repository

Step-2: Copy the repository url from the forked repo

Step-3: Open terminal and write git clone `url you just copied`

Step-4: In terminal write cd supreme-lamp

Step-5: Write git checkout -b `any desired word/words`

Step-6: Contribute to the code of this webpage

Now edit the code Update Ui-or add more fetures etc .
Use your creativity here.

Step-7: In the terminal check git status then add the files you changed using git add `modified\add file name`

Step-8: Write in the terminal, git commit -m `Your Message`

Step-9: Write in the terminal, git push origin `Name you used in Step-4`
